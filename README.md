# Web Development Assignment - HTML Exercises

## Description

This project is a comprehensive HTML assignment that demonstrates fundamental web development concepts. The assignment includes practical exercises covering:

- **Basic Web Page Structure** - Headings, paragraphs, lists, links, and images
- **HTML Forms** - Text inputs, radio buttons, checkboxes, textareas, and form submission
- **HTML Tables** - Structured data presentation with tables
- **Semantic HTML5 Elements** - Proper use of semantic markup

The project showcases how to build a simple, well-structured webpage using core HTML elements and basic inline CSS styling.

## Steps to Run the Project

1. **Clone or Download** the project to your local machine

2. **Navigate** to the project folder:
   ```
   cd Web_dev Assignment/Html_assignment1
   ```

3. **Open the HTML file** in a web browser:
   - Double-click on `practice1.html`, OR
   - Right-click → Open with → Choose your preferred browser, OR
   - Use VS Code Live Server extension for live preview

4. **View the webpage** - The browser will display all the HTML exercises including:
   - A simple webpage with headings, lists, and images
   - A contact form with various input types
   - HTML tables demonstrating data structure
